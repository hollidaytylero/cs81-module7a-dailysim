# cs81-module7a-dailysim
Module 7a Daily Simulation

Overview
In this assignment, you'll simulate a personalized or fictional daily routine using setTimeout to control when each activity is shown on the screen. You'll practice sequencing asynchronous events and reflect on how well your code simulates real time.

Learning Goals
Practice using setTimeout for sequencing events
Understand the flow of asynchronous code
Build meaningful, unique sequences with your own logic
Connect code behavior to real-world timing
Reflect on asynchronous challenges and design decisions
Tasks
Complete and customize the starter code in dailySimulation.html
Add at least 6 personal or creative activities with timing
Add one surprise or twist (random time, mood, or emoji)
Write and complete REFLECTION.md
Commit your work and submit to Canvas and GitHub
Make at least 4 Git commits:
Initial structure
Simulation events
Enhancement or twist
Final polish and reflection
